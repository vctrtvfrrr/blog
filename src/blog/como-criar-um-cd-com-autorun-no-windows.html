--- title: Como Criar um CD com Autorun no Windows date: 2007-10-24T22:17:47-03:00 description: Aprenda a criar um CD com autorun no Windows abstract: Aprenda a criar um CD com autorun no Windows tags: [windows, cd, autorun] ---

Já pensou em criar um CD de instalação de algum programa, ou um arquivo batch autoexecutável no Windows? O que você acha de ao colocar o seu CD na unidade ter imediatamente iniciado o seu sistema de instalação ou executar seu script batch?

Bem, a coisa é tão simples que relutei em inserí-la aqui como uma dica. Mas é basicamente o seguinte:

Criar um arquivo chamado AUTORUN.INF com as seguintes linhas:

[autorun]
OPEN=SETUP.EXE /AUTORUN

Onde SETUP.EXE é o nome do arquivo que você deseja executar automaticamente. Se você deseja executar um arquivo batch, por exemplo, basta substituir SETUP.EXE pelo nome do arquivo batch.

Depois, basta gravar os arquivos AUTORUN.INF e o executável para a raiz do CD-ROM. Pronto! Ao inserir o CD na unidade, o arquivo será executado automaticamente.

Por hoje é só isso... tchau!
